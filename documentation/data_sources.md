# Data Sources

This dashboard uses public data from the European Central Bank data portal.

## Sources Used

- EUR/USD exchange rates
- ECB deposit facility rate
- Euro area inflation rate
- Euro area lending rate

## Data Access Method

Data is accessed directly through ECB API endpoints using Power Query in Power BI Desktop.

## Refresh

The report is published to Power BI Service and configured with a daily scheduled refresh.
